# IT Salary Analytics Dashboard (2026)

This Power BI dashboard analyzes salary distribution across job roles, experience levels, and locations.

## KPIs
- Total Salary (LPA)
- Average Salary (LPA)

## Visualizations
- Pie Chart – Salary Distribution by Job Role
- Bar Chart – Salary Comparison by Experience Level
- Donut Chart – Salary by Location
- Slicers – Job Role, Level, Location

## Tools Used
- Power BI
- Excel
